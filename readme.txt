== Installation ==

Aqui estão os passos para instalar e configurar o seu plugin:

    1. **Instale** e **ative** o plugin, da forma como preferir;
    2. Configure seu plugin com os dados da sua loja em **Elementos Auryn**;
       - **Domínio da empresa** - Caso a sua empresa tiver um domínio próprio (deve ser loja Auryn);
       - **É o Wordpress principal da Auryn** - Apenas para configuração pela equipe da Auryn
    3. Salve as suas alterações;
    4. Pronto, agora é só utilizar os widgets dentro do Elementor.
 
== Frequently Asked Questions ==
 
= Não está funcionando =
 
Verifique algumas questões:

    1. O **Elementor está instalado?**
    2. Foi configurado o seu domínio corretamente, nas configurações?
       - Ex: **modelo.auryn.com.br**
    3. A configuração **É o Wordpress principal da Auryn** está desativada?
